## üìö Featured Project: Text Analysis & AI Humanizer

A comprehensive Python tool for bias detection, AI text humanization, professional formatting, and text quality analysis.

### üéØ What Does It Do?

This tool helps you:
1. **Detect Bias** - Find gender, racial, age, political, ability, and socioeconomic bias in writing
2. **Humanize AI Text** - Transform AI-generated text into natural, human-like writing
3. **Format Documents** - Convert text to MLA, essay, letter, cover letter, or resume format
4. **Analyze Quality** - Get readability scores, word counts, and quality assessments

### üöÄ Quick Start

#### Option 1: Interactive Mode (Recommended for Beginners)

```bash
# Run the interactive wizard
python main.py
```

The interactive mode will guide you through:
1. Pasting your text
2. Selecting features (bias detection, AI humanization, formatting)
3. Choosing format options
4. Viewing and saving results

#### Option 2: Command-Line Mode

```bash
# Analyze a file for bias and humanize it
python main.py -f input.txt -b -h -o output.txt

# Format as cover letter
python main.py -f letter.txt -h -t cover_letter -o formatted_letter.txt

# Full analysis with all features
python main.py -f essay.txt -b -a -h -t mla -o final_essay.txt
```

**Command-Line Options:**
- `-f FILE` or `--file FILE` - Input file to process
- `-b` or `--bias` - Detect bias in text
- `-a` or `--ai-analysis` - Analyze AI likelihood
- `-h` or `--humanize` - Humanize AI-generated text
- `-t TYPE` or `--format-type TYPE` - Format type: mla, essay, letter, cover_letter, resume, plain
- `-o FILE` or `--output FILE` - Save results to file

#### Option 3: Python API

```python
from main import TextProcessingApp

# Create app instance
app = TextProcessingApp()

# Your text
text = """
As an AI language model, I want to discuss climate change.
It is important to note that we must take action now.
"""

# Configure options
options = {
    'detect_bias': True,      # Check for bias
    'analyze_ai': True,       # Analyze AI probability
    'humanize': True,         # Make text more human
    'format_type': 'essay',   # Format as essay
    'format_params': {
        'title': 'Climate Change',
        'author': 'Your Name'
    }
}

# Process the text
results = app.process_text(text, options)

# Generate report
report = app.generate_report(results, options)
print(report)
```

#### 1. bias_detector.py - Bias Detection Engine

**What it does:** Scans text for biased language across 6 categories.

**How it works:**

```python
class BiasDetector:
    def __init__(self):
        # Stores dictionaries of biased terms by category
        self.gender_biased_terms = {...}
        self.racial_bias_terms = [...]
```

**Detection Process:**
1. **Pattern Matching** - Compares text against database of biased terms
2. **Pronoun Analysis** - Checks for gender pronoun imbalance (he/she ratio)
3. **Context Analysis** - Identifies stereotypical phrases
4. **Severity Scoring** - Rates each bias as low/medium/high severity
5. **Scoring** - Calculates overall bias score (0-100)

**Example Detection:**

**Readability Formulas:**

**Flesch Reading Ease:**
```python
# Formula: 206.835 - 1.015(words/sentences) - 84.6(syllables/words)
# Score: 0-100 (higher = easier to read)
# 90-100: 5th grade level
# 60-70: 8th-9th grade level
# 0-30: College graduate level
```

**Flesch-Kincaid Grade Level:**
```python
# Formula: 0.39(words/sentences) + 11.8(syllables/words) - 15.59
# Returns US grade level required to understand text
# Example: 8.5 = 8th grade, halfway through year
```

**Syllable Counting Algorithm:**
```python
def _count_syllables(self, word):
```
